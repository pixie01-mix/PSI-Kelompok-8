# Library Management System

Project Pengembangan Sistem Informasi menggunakan Agentic AI.

## Deskripsi
Project ini merupakan simulasi Library Management System sederhana yang dikembangkan menggunakan bantuan AI Agent (Codex/Claude).

## Fitur
- Registrasi & Login pengguna
- Manajemen data buku
- Sistem Peminjaman buku
- Dashboard admin
- Riwayat peminjaman	

## Teknologi
- Node.js
- Express.js
- HTML CSS JavaScript
- Agentic AI

## Cara Menjalankan

### Backend
cd backend
npm install
node server.js

### Frontend
Buka file index.html menggunakan browser.
